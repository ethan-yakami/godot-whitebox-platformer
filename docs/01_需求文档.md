# 白盒原型需求文档

## 项目目标

制作一版 Godot 4.x 的 2D 横板平台跳跃白盒原型。玩家一周目从起点出发，通过 1-5 关获得传承；二周目携带传承返回起点。首版目标是验证“按键次数限制 + 去返程 + 死亡残影”的核心玩法闭环。

## 已确认规则

- 左/右限制指 A/D，不是鼠标左键/右键。
- 冲刺前期不可用，第五关获得传承后解锁。
- 第一关通关后解锁二段跳。
- 死亡回最近路灯，有读档点就复活到读档点。
- 返程死亡会在死亡点生成残影，最多 2 个；首版残影为死亡点附近敌对巡逻/短距离追击。
- 第一版只做白盒，不追求正式美术。

## 操作

| 输入 | 功能 |
| --- | --- |
| A | 向左移动 |
| D | 向右移动 |
| W | 视野上移预留 |
| S | 视野下移预留 |
| H | 跳跃；解锁后二段跳 |
| Space | 冲刺；获得传承后可用 |
| J | 攻击/交互 |
| Esc | 暂停 |

## 核心循环

1. 玩家进入当前关卡。
2. 关卡加载该阶段的按键次数限制。
3. 玩家规划路线，利用有限操作通过平台、机关、敌人。
4. 到达出口进入下一关或返程上一关。
5. 第五关获得传承后切换返程。
6. 返程死亡生成残影并回最近路灯。
7. 携带传承回到第一关起点，游戏胜利。

## 生命与能力

- 初始生命上限 3。
- 获得传承后生命上限变为 5，并解锁冲刺。
- 尖刺、移动尖刺扣 1 血。
- 特殊障碍地形扣 2 血。
- 怪物攻击扣 1 血。
- 浆果回复 1 血。
- 路灯作为自动存档点，进入光线范围后回满状态并记录复活点。

## 关卡限制

| 关卡 | 一周目限制 | 二周目限制 | 奖励 |
| --- | --- | --- | --- |
| 1 草原 | 跳跃 x5 | A 左移 x1 | 二段跳 |
| 2 湖泊 | 跳跃 x9 | A 左移 x5 | 无 |
| 3 森林 | A 左移 x3 | 跳跃 x5 | 无 |
| 4 溶洞 | D 右移 x3 | A 左移 x5 | 攻击键教学 |
| 5 古塔 | A 左移 x4；Boss 战后解除可后续优化 | D 右移 x2 | 传承、冲刺、5 血 |

## 胜负条件

- 胜利：返程阶段持有传承并回到起点。
- 死亡：生命为 0 后回最近路灯；返程额外生成残影。
- 第一版不做整局失败结算，避免影响 Game Jam 调试节奏。
