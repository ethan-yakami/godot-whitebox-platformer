# 关卡拖拽编辑指南

## 打开与修改

1. 用 Godot 打开项目目录：`C:\Users\user\Desktop\游戏开发`。
2. 在 FileSystem 面板打开 `levels/level_1.tscn` 到 `levels/level_5.tscn`。
3. 展开节点分组：
   - `Blocks`：平台、移动板、塌陷平台、弹跳垫。
   - `Hazards`：尖刺、水流/红色危险槽。
   - `Markers`：出生点、返程出生点、路灯、出口、石碑、献祭石。
   - `Actors`：敌人、Boss。
   - `Guides`：策划图起始视野框参考，不参与碰撞。
4. 直接选中节点后拖动位置。需要改尺寸时，在 Inspector 修改导出变量，例如 `size`、`count`、`offset`、`speed`、`push_force`。

## 通关距离约束

- 普通跳舒适高度：`120-160px`，极限不要超过 `180px`。
- 二段跳舒适高度：`220-300px`。
- 普通跳横向距离：`180-260px`，极限不要超过 `320px`。
- 移动平台速度建议：`90-130px/s`。
- 移动平台行程建议：`160-260px`。

## 组件说明

- `WhiteboxPlatform`：灰色静态平台，改 `size`。
- `SpikeStrip`：蓝色尖刺，改 `count` 和 `direction`。
- `WaterZone`：浅黄水流或红色危险槽占位，改 `size` 和 `push_force`。
- `MovingPlatform`：橙色移动平台，改 `size`、`offset`、`speed`。
- `CollapsePlatform`：粉色塌陷平台，改 `size`、`collapse_delay`。
- `BouncePad`：绿色弹跳垫，改 `bounce_force`。
- `Checkpoint`：黄色路灯，玩家进入即设为复活点。
- `ExitGate`：绿色出口，玩家进入自动切关。
- `Stone`：第四关石碑，靠近按 `J` 解锁攻击。
- `Altar`：第五关献祭石，Boss 击败后按 `J` 获得传承。

## 改完后检查

改动关卡后至少跑：

```powershell
godot --headless --path . --quit
godot --headless --path . --quit-after 2
```

如果只是微调位置，还需要人工从起点跑到终点，确认限制次数仍然够用。
