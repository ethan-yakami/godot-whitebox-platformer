# 白盒关卡主通关路线

- 第一关：按策划标注的 5 段跳跃设计。第 1 跳到低浮空灰台，第 2 跳到粉色塌陷小台，第 3 跳到中段粉色平台，第 4 跳必须在空中调整二段跳时机，避开上方灰色阻挡块后落到高台，第 5 跳到终点。示意图不是实际比例，场景坐标按当前玩家跳高重新调过。
- 第二关：按两张 PPT 合并为 Scene1 + Scene2。不是所有红线路径都消耗跳跃，部分段落依赖下落、高低差、移动平台横向惯性、水流向右推力。总跳跃预算仍按 x9，但主路线允许利用 DropMomentum / NoJumpGlide 这类非跳跃段过渡。
- 第三关：主路线主要向右推进，A左移只用于落点微调。
- 第四关：先到右侧石碑解锁攻击，再处理敌人并上到终点。
- 第五关：前段低天花板通过移动平台，击败Boss解除限制，右侧台阶到献祭石。

跳距预算：普通跳舒适横距180-260px，普通跳高度120-160px，二段跳高度220-300px，移动平台速度约110px/s。
