# 策划案（6。19）.docx

## 段落
1. [Heading 1] 一、游戏特色
2. [Normal] 这是一款以冒险为主题的2D横板平台跳跃游戏。玩家需要通过重重陷阱和复杂地形前往目的地获得传承，再返回起点利用传承解救即将被灾难摧毁的王国。
3. [Normal] 游戏的核心特色在于限制操作键使用次数，一周目为前往传承的旅途。每一关卡设有不同的按键限制，如跳跃、移动或技能，都可能成为本关的稀缺资源。玩家需要审时度势，精打细算地规划路线与操作，才能在资源耗尽前抵达终点获得传承。二周目为返程，玩家每回合死亡后生成残影干扰玩家行动，残影具备与玩家相同的技能，且可叠加至上限两个，死亡次数越多，归途障碍越多。
4. [Heading 1] 二、设定简要
5. [Heading 2] 2.1 行动规则
6. [Normal] 玩家出生于王国某处，需操控角色穿越多个相互连通的关卡地形，最终抵达传承所在地。
7. [Normal] 玩家的行动受到按键限制约束，每个关卡限制的按键类型不同。玩家需根据当前关卡的限制合理规划路线与操作，在次数耗尽前通过关卡。
8. [Normal] 游戏分为去程与返程两个阶段：
9. [Normal] 1、去程：玩家正常通关，收集传承。
10. [Normal] 2、返程：玩家携带传承踏上归途。每次死亡会在死亡位置生成一道残影，残影徘徊于原地并拥有与玩家相同的技能。残影可叠加，上限为2个。死亡次数越多，归途中的障碍越多。
11. [Heading 2] 2.2 获胜条件
12. [Normal] 游戏的获胜条件为：玩家完成去程取得传承后，在返程中成功穿越所有关卡，携带传承回到起点。玩家需同时满足两个条件：一是持有传承，二是成功抵达起点。若玩家在返程途中因残影或地形陷阱导致体力耗尽，则本局游戏失败。
13. [Heading 2] 2.3 游戏界面
14. [Heading 3] 2.3.1 游戏大厅界面
15. [Normal] 玩家登陆游戏后，首先进入游戏大厅。
17. [Normal] 图2.1 游戏大厅界面示意图
18. [Normal] （1）开始游戏
19. [Normal] 按钮作用：
20. [Normal] 用于进入正式游戏流程，是大厅页中最核心的功能按钮，应在视觉上最突出。点击后进入游戏。
21. [Normal] （2）设置
22. [Normal] 按钮作用：用于调整游戏基础配置，让玩家根据自己的习惯进行设置。
23. [Normal] 点击后内容：点击后弹出设置面板，包含基础设置项：
24. [Normal] 音量调节：调整背景音乐和音效大小； 画面设置：调整全屏/窗口模式； 操作说明：查看移动、查看监控、互动等按键； 返回按钮：关闭设置面板并回到大厅页。
25. [Normal] （4）退出游戏
26. [Normal] 按钮作用：用于退出当前游戏或关闭游戏程序。
27. [Normal] 点击后内容：点击后弹出二次确认弹窗，避免玩家误触。弹窗内容可以是：“确定要离开吗？”按钮包括：“确认退出”和“取消”。

34. [Normal] 2.3.3 游玩内容
35. [Normal] 1、玩家操作与角色能力
36. [Normal] 跳跃：分为单次跳跃和二段跳，二段跳在限制跳跃的关卡中算两次跳跃。
37. [Normal] 移动：左右移动，上下可往上下移动一段距离的视野。
38. [Normal] 技能：冲刺，可往目前前进方向快读移动一段距离，cd冷却时间为0.5s
40. [Heading 2] 2.4 游戏操作方法
41. [Normal] 游戏采用鼠标+键盘的操作方式，尽量简化了操作难度。
43. [Heading 2] 2.5 道具设定
44. [Normal] 游戏对局中，当玩家移动至可交互型道具较近范围内时，该道具四周边框会呈高亮可交互状态，玩家可使用交互键进行交互，并产生对应效果。
46. [Heading 1] 三、关卡设计和战斗系统
47. [Heading 2] 3.1 关卡场景
48. [Normal] 游戏包含多个关卡，关卡中分布着不同的道具和可交互物品，详见附件PPT。
50. [Normal] 3.3 战斗系统设定
51. [Normal] 3.3.1玩家角色设定
52. [Normal] 血条
53. [Normal] 常规状态：初始状态为3滴血，获得传承后增至5滴血
54. [Normal] 扣血：被尖刺、移动尖刺扣一滴血，掉入特殊障碍地形扣2滴血，每被怪物攻击一次扣1滴血。
55. [Normal] 回血：食用浆果回复1滴血，在存档路灯光线下1s回复1滴血。
56. [Normal] 攻击力：一次攻击扣除两滴血
